# X-Ray Mod
Minecraft Forge XRay mod

### Supports
- Minecraft 1.7.10      / 1.9.4
- Forge 10.13.2.1291    / 12.17.0.2051

### Feature:
- Full Support for Forge
    - Supports versions back from 1.7.10
- Clean and simple GUI to add blocks
- Colour RGB Selector
- Auto adds ores from other mods.
